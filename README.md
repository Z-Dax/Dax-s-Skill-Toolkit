## About Daily-News-Scan Skill
> 一个为深度信息工作者打造的「全球科技 / AI / 商业资讯每日扫描」Skill。
> A daily intelligence-gathering Skill for deep knowledge workers.
---

### 一句话定位
> **Daily News Scan** —— 把每天 3 小时的"刷资讯"，压缩成 3 分钟的"读简报"。

### 它是什么
基于一份 **200+ 条、经过 ⭐1–5 星可靠度评分** 的可信信源清单,按"快讯 → 行业 → 深度 → 研究"四层阅读策略自动扫描、去重、聚类,最终产出一份 **结构化、可追溯、零水分** 的每日简报。

### 六大亮点

| 亮点 | 说明 |
|---|---|
| 🌐 **200+ 信源全景覆盖** | 综合科技、AI 垂直、商业财经、研究智库、Newsletter、播客、YouTube、X 博主、中文媒体——十大类一网打尽 |
| ⭐ **每一条都带可靠度评分** | 从通讯社(Reuters / Bloomberg)到独立 Newsletter(Stratechery / Import AI),1–5 星透明打分,让你知道在读什么 |
| 🪜 **L1–L4 分层阅读策略** | 快讯层 → 行业层 → 深度层 → 研究层,宽度优先再下钻,避免信息过载 |
| 🧠 **自动去重 + 多源聚类** | 同一事件 5 家媒体报道?自动合并成 1 条,并列出所有信源 |
| 🔒 **付费墙优雅处理** | The Information / FT / Stratechery 等付费内容会被标记 🔒,保留标题不丢线索 |
| 🛠 **可拼装的工作流** | 一键输出到飞书文档、推送到飞书群、归档到多维表,或交给 Mira Scheduler 每日定时跑 |

### 适合谁用
- 🧑‍💻 **AI / 科技从业者**:每天需要跟进行业但没时间逐站点刷
- 📈 **投资人 / 战略分析师**:要捕捉早期信号(潜在信号区专门为你设计)
- ✍️ **内容创作者 / 自媒体**:找选题、追热点、抢首发
- 🧠 **信息焦虑党**:受够了"震惊体"和算法投喂

### 怎么用(零学习成本)
只需一句话:

> "帮我做一份今天的 AI 简报,覆盖中美,包含深度文章。"

Skill 会自动完成:**选信源 → 并行抓取 → 去重聚类 → 提炼信号 → 生成简报**。

支持的参数(不填用默认值):

- `scope` 主题范围 · `time_window` 时间窗 · `depth` 阅读层级
- `region` 区域偏好 · `min_reliability` 可靠度下限 · `output` 输出格式

### One-liner

> **Daily News Scan** — Turn 3 hours of doom-scrolling into a 3-minute briefing.

### What it is

A daily intelligence-gathering Skill powered by a **curated list of 200+ trusted sources** — each rated on a transparent **1–5 star reliability scale**. It scans, deduplicates, and clusters global tech / AI / business news using a four-tier reading strategy (**L1 Newswire → L2 Industry → L3 Deep-dive → L4 Research**), and produces a clean, structured briefing you can actually act on.

### Six Highlights

| Highlight | Description |
|---|---|
| 🌐 **200+ sources, fully mapped** | Mainstream tech, AI verticals, business & finance, research institutes, newsletters, podcasts, YouTube, X influencers, Chinese media — 10 categories, fully covered |
| ⭐ **Every source rated** | From wires (Reuters, Bloomberg) to indie newsletters (Stratechery, Import AI) — transparent 1–5 star reliability, so you always know what you're reading |
| 🪜 **L1–L4 layered reading** | Breadth-first, then drill down. No more drowning in headlines |
| 🧠 **Auto dedup & cross-source clustering** | Same story covered by 5 outlets? Auto-merged into one item with all sources listed |
| 🔒 **Graceful paywall handling** | Paywalled content (The Information / FT / Stratechery) is flagged, never silently dropped |
| 🛠 **Composable workflow** | Export to Lark Doc, push to a Lark group, archive into Bitable, or schedule daily via Mira Scheduler |
